# EOnline

一個可透過 Discord 機器人遊玩的現實生活模擬器專案，使用 Rust 開發。

## 目標

- 建立可互動的 Discord 遊戲體驗
- 模擬現實生活中的角色狀態、行動與事件
- 提供可擴充的遊戲系統（經濟、任務、成長等）

## 開發

### 需求

- Rust（建議最新 stable）
- Cargo

### 執行

```bash
cargo run
```

### 測試

```bash
cargo test
```
