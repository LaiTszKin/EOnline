# Contributing

感謝你對 EOnline 的關注！

## 基本流程

1. Fork 本專案
2. 建立功能分支：`git checkout -b feature/your-feature`
3. 提交修改：`git commit -m "feat: your feature"`
4. 推送分支：`git push origin feature/your-feature`
5. 開立 Pull Request

## 開發規範

- 保持程式碼可讀性與模組化
- 新功能請附上適當測試（若適用）
- Commit 訊息建議使用語意化前綴（如 `feat:`, `fix:`, `chore:`）
